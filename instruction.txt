
Инструкция по запуску тестов.

1. Запуск phonebook.jar посредством командной строки: java -jar phonebook.jar
2. Открываем проект testPhonebook с помощью IntellijIDEA
3. С помощью раздела Maven запускаем тесты командой mvn test
4. Результат прохождения тестов появится внизу окна, помимо этого отчет о прохождении тестов можно
посмотреть в сгенерированном файле testLogReport.log


В коде содержатся комментарии о назначении каждого теста.
Тест negTestEmail служит для демонстрации найденой баги. Описание баги приведено в файле bugReport.txt.

В файле testplan.txt описан созданный мною тест-план, на который я ориентировалась при написании автотестов.
